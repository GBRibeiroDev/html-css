grouping-tags

vamos ver algumas tags semânticas, começando pela header que utilizamos para fazer cabeçalhos
exemplo de site semântico: 
<header>

    </header>
    <main>

    </main>
    <footer>
        
    </footer>

dentro do header nós podemos colocar um h1/logo-marca. em seguida podemos criar o menu utilizando a tag nav e dentro dela colocando os links.
dentro da tag main que é nosso conteúdo principal, podemos criar algumas sections. 
dentro dessas sections que nós criamos os article para falarmos dos artigos.
e dentro do article podemos colocar também o conteúdo periférico utilizando o aside.


explicação de outro aluno: 
 Tags de agrupamento (grouping tags):

- Sem valor semântico:

   <div>		-> block
   <span>	-> inline

- Com valor semântico:

  <header>		-> cabeçalho
       <nav>	        -> para menus, links, navegação

 <main>			       -> principal
   <section>		       -> seção
	 <article>	       -> artigo
		 <aside>	       -> periférico ao artigo (ex: escritor / autor)

 <footer>	               -> rodapé
